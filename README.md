# LuatOS For ESP32 And PlatformIO

## 1、简介

为了方便代码编译、环境统一、以及跨平台开发。

使用PlatformIO平台进行 ESP32 [LuatOS](https://gitee.com/openLuat/LuatOS) 项目构建。

已经适配ESP32

ESP32S2,ESP32C3正在加紧适配中
